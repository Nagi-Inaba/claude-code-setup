# LP Content Template (mdC) — 可変入力テンプレート

## 基本情報

| 項目 | 値 | ガイド |
|------|---|-------|
| `{{company_name}}` | | 会社名/サービス提供者名 |
| `{{product_name}}` | | プロダクト/サービス名 |
| `{{product_url}}` | | プロダクトURL（CTAリンク先） |
| `{{logo_url}}` | | ロゴ画像URL（なければテキストロゴ） |
| `{{industry}}` | | 業種（ui-ux-pro-max のスタイル決定に使用） |

## ヒーロー

| 項目 | 値 | ガイド |
|------|---|-------|
| `{{tagline}}` | | キャッチコピー（15文字以内推奨） |
| `{{subtitle}}` | | サブコピー（30-50文字、課題→解決を1文で） |
| `{{cta_primary}}` | | 主CTAボタンテキスト（例: 「無料で始める」「資料請求」）|
| `{{cta_secondary}}` | | 副CTAテキスト（例: 「デモを見る」「詳しく見る」）|
| `{{trust_badges}}` | | 信頼指標（例: 「導入500社」「満足度98%」）カンマ区切り |
