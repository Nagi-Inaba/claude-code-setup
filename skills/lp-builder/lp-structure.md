# LP Structure (mdB) — 11セクション構成

## セクション一覧

### 1. ヘッダー
- **レイアウト**: 固定ナビ（sticky top）
- **必須要素**: ロゴ（左）、ナビリンク（中央）、CTAボタン（右）
- **レスポンシブ**: モバイルでハンバーガーメニュー
- **高さ**: 64px (PC), 56px (モバイル)
- **背景**: スクロールで `backdrop-blur + bg-white/90`

### 2. ヒーロー
- **レイアウト**: 2カラム（テキスト左 60% + ビジュアル右 40%）、モバイルでスタック
- **必須要素**: キャッチコピー(H1)、サブコピー(Body Large)、CTAボタン(Primary)、サブCTA(Secondary)
- **任意**: 信頼バッジ（「導入○社」「満足度○%」）
- **高さ**: min-height 80vh (PC), auto (モバイル)
- **背景**: グラデーションまたは単色 `--color-bg-alt`

### 3. 課題提起
- **レイアウト**: 3カラムカードグリッド、モバイルで1カラム
- **必須要素**: セクション見出し(H2)、3つの課題カード（アイコン + タイトル + 説明文）
- **カード仕様**: Default Card、アイコンは絵文字またはSVG
- **背景**: `--color-bg`

### 4. ソリューション
- **レイアウト**: 交互配置（テキスト＋画像が左右交互）、モバイルでスタック
- **必須要素**: セクション見出し(H2)、3-5つの機能紹介（H3 + 説明文 + スクリーンショット/イラスト）
- **任意**: 各機能にミニCTA
- **背景**: `--color-bg-alt`

### 5. 特長
- **レイアウト**: 3カラムアイコングリッド
- **必須要素**: セクション見出し(H2)、3-6つの特長（大アイコン + H3 + 短い説明文）
- **アイコンサイズ**: 48px
- **背景**: `--color-bg`

### 6. 導入実績/数字
- **レイアウト**: 4カラム統計カウンター、モバイルで2x2グリッド
- **必須要素**: セクション見出し(H2)、3-4つの数値（大数字 + ラベル）
- **アニメーション**: IntersectionObserver でカウントアップ
- **背景**: `--color-secondary` + white テキスト（ダークセクション）

### 7. 導入事例/お客様の声
- **レイアウト**: カルーセル（PC: 3枚表示）またはグリッド
- **必須要素**: セクション見出し(H2)、3-6つの声（顔写真 + 名前 + 肩書 + コメント）
- **任意**: 会社ロゴ、星評価
- **背景**: `--color-bg-alt`

### 8. 料金プラン
- **レイアウト**: 3カラム比較表、中央にハイライト
- **必須要素**: セクション見出し(H2)、3プラン（名前 + 価格 + 機能リスト + CTAボタン）
- **ハイライト**: Highlighted Card（border-top primary）+ 「おすすめ」バッジ
- **任意**: 年払い/月払いトグル、FAQ リンク
- **背景**: `--color-bg`

### 9. FAQ
- **レイアウト**: アコーディオン（最大幅 800px、中央揃え）
- **必須要素**: セクション見出し(H2)、5-8つのQ&A
- **開閉**: `<details><summary>` または JS アコーディオン
- **背景**: `--color-bg-alt`

### 10. CTA（最終コンバージョン）
- **レイアウト**: 全幅バナー、中央揃え
- **必須要素**: キャッチコピー(H2)、サブコピー、CTAボタン(Primary, 大サイズ)
- **任意**: フォーム埋め込み（メールのみ）、「無料で始める」テキスト
- **背景**: `--color-primary` のグラデーション + white テキスト

### 11. フッター
- **レイアウト**: 4カラム（会社情報 + リンク群 x 3）、モバイルでスタック
- **必須要素**: ロゴ、会社情報、リンク群、コピーライト
- **法的情報**: プライバシーポリシー、特定商取引法、利用規約へのリンク
- **背景**: `--color-secondary` + muted テキスト

## セクション順序ルール

- ヒーロー → 課題 → ソリューション の流れは固定（AIDA: Attention → Interest → Desire）
- 料金は FAQ の前（価格疑問を FAQ で解消）
- 最終 CTA は必ず FAQ の直後
- ヘッダーとフッターは常に最上部/最下部

## 省略可能セクション

料金プランがない場合（問い合わせ型）:
- セクション8を省略
- セクション10のCTAを「お問い合わせフォーム」に変更

| 名前 | 幅 | カラム数 |
|------|---|---------|
| Mobile | < 640px | 1 |
| Tablet | 640px - 1024px | 2 |
| Desktop | > 1024px | 3-4 |
