# rules/ — コーディングルール

このディレクトリには言語共通ルールと言語別ルールが含まれる。

## 構造

| ディレクトリ | 対象 |
|-----------|-----|
| `common/` | 全言語共通（コーディングスタイル・テスト・セキュリティ等） |
| `typescript/` | TypeScript/JavaScript 固有ルール |
| `python/` | Python 固有ルール |
| `golang/` | Go 固有ルール |
| `kotlin/` | Kotlin/Android 固有ルール |
| `swift/` | Swift 固有ルール |
| `php/` | PHP 固有ルール |
| `perl/` | Perl 固有ルール |

## 優先順位

言語別ルール > common ルール（言語固有が優先）
